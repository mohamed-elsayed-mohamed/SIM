Managers: A/B — vcpkg manifest · C/D — NuGet · E — UPM

Build pipeline workflow — resolve → compile → publish

① Resolve dependencies

vcpkg install (A, B) · dotnet restore (C, D) · UPM when Unity runs

→
② Compile

MSBuild A→B · dotnet C→D · Unity CLI E if Editor present

→
③ Publish

Folder + manifest.json · CI: dotnet pack + NuGet push

Artifact targets after publish

Local / Python

artifact_repo/Python/<version>_<timestamp>/

CI folder

artifacts/<run_number>/

GitHub Packages

SIM.ProjectC, SIM.ProjectD

Optional feed

NUGET_FEED_URL + NUGET_FEED_API_KEY

Python entry — build.py stage tree

python Build/Python/build.py --config Release --version 1.0.0
        │
        ├─ [load_config]       build_config.json + CLI (CLI overrides file)
        │
        ├─ [detect_tools]        MSBuild · vcpkg · dotnet · Unity (optional)
        │
        ├─ [resolve_deps]        vcpkg install · dotnet restore C,D
        │
        ├─ [compile]             order A → B → C → D → E
        │
        └─ [publish]             artifacts/Python/<version>_<timestamp>/ + manifest.json
